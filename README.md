# Safepaws
Web application dedicated to cat rescue efforts. Main features include an interactive map and adoption page.

# Getting started
The following instructions should help you run this project on your local device.

Installation:
1. Clone the repository.
2. Install dependencies:
- You will need to install Python and Node.js.
After installing Node.js, run 'npm install' and 'npm start' in the terminal of the frontend directory to start the frontend server.
To run the backend, you should run 'python -m uvicorn app.main:app --reload' in your terminal.
- You will also need a Mapbox GL JS API key.
3. Adjust environment variables such as API key and API base URL.

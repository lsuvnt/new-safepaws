# Safepaws
Web application dedicated to cat rescue efforts. Main features include an interactive map and adoption page.

# Getting started
The following instructions should help you run this project on your local device.

# Dependencies
1. Node.js
2. Python
3. Mapbox GL JS API key

# Installation
1. Clone the repistory.
2. Install dependencies
   Node.js: run 'npm install' and 'npm start' in terminal
3. Adjust environment variables such as API key.
